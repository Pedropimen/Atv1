Html

A parti da aula foi visto que html não é uma linguagem de promação mas sim uma linguagem de marcação tendo suas marcações feitas por ''tags''
Como por exemplo:

  • a,href,p,

  • h1,h2,h3,h4....

  • article,div,section...

existindo varios modelos do html que foram evoluindo com o tempo sendo o ultimo modelo o HTML:5.Concluido temos varios tipos de tags, sendo html uma linguagem de marcação completa,com a mais recente html:5 podendo ter uma variedade de possibilidade de tabelas, textos, imagens

Outros Assuntos vistos

JavaScript (JS)

O Java Script diferentemente do html já não é uma linguagem de marcação mas sim uma linguagem de programação usada para atividades dinâmicas no site,
executando funções em formato de codigo exercendo uma operação, como abrir outra pagina ou direcionamento de dados ao banco de dados

CSS

O CSS diferentemente dos outros 2 tipos é uma linguagem de formatação, onde por meio dele podemos adicionar ''adjetivos'', caracteriscas a pagina, como cor de fundo, cor as letras, estilos de letras formatação dos texto entre outras, também sendo divido em tipos de ''tags''

Como por Exemplo
  • Backgroudcolor,color (referente as cores)

  • width,lef,top,right(referente aos lados)

  • fixed,solid,border(referente a uma caracterisca dada ao item selecionado
